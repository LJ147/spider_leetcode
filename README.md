# spider_leetcode
基于scrapy的LeetCode查卡代码段

# 一、使用方法

1.安装依赖库：`pip install scrapy`

2.进入项目根目录之后 , 继续 `cd spider_leetcode`

3.执行 `scrapy scrapy submission`
